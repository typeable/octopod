Keys and values

air.vaultEngineName:


Классы

Ключ по умолчанию, значение не изменено: key-default-pristine value-pristine
Ключ по умолчанию, значение изменено: key-default-edited value-edited
Ключ кастомный, значение изменено: key-custom-edited value-edited
Ключ и значение удалены: key-deleted value-deleted
Ключ удалён, значение неизвестно: key-deleted unknown (value-unknown)


Примеры

Ключ по умолчанию, значение не изменено: air.vaultEngineName: staging
Ключ по умолчанию, значение изменено: air.vaultEngineName: staging
Ключ кастомный, значение изменено: app_api_url: http://my-api.my-domain.com
Ключ и значение удалены: sso.vaultEnv.hotels-sso.mapping[0].APP_URL http://my-api.my-domain.com
Ключ удалён, значение неизвестно: air.vaultEnv.air-sso: unknown


Для экономии узлов в DOM-дереве можно не создавать отдельную обёртку <span>...</span> а вешать эти классы прям на ячейки таблицы, или на любого другого родителя. Работает со всеми классами кроме .value-unknown -- это отдельный компонент.
